Google Ads Search-Campaign Reporting Script

This project provides a single, read-only Google Ads Script that creates a fully formatted, multi-tab Google Sheet in Drive—no APIs, add-ons, or manual spreadsheet work required.

What you get

Sheet tabHighlights
READMEPurpose of each tab • colour legend • disclaimer
CONFIGEdit thresholds without touching code
OverviewClicks · CTR · CPC · CPA · ROAS (currency auto-detected)
HeatmapDay × Hour grids (Clicks · CTR · CPC · Conversions · CPA)
StrategyRule tips • device CPA gaps • live Google-Ads recommendations
AdsEnabled ads — colour-coded Keep / Test / Pause with advice
Campaign-KWFull keyword list, Quality Score, Top-10 & Bottom-10 sections, actions
Search TermsBrand-aware Add / Review / Exclude

Quick Start (3 steps)

  1. Open Google Ads → Tools → Bulk actions → Scripts and paste search-report.gs.
  2. Replace SPREADSHEET_ID (top of the script) with the ID of a blank Sheet.
  3. Click Preview → Authorise → Run once, then schedule (e.g. 06:00 • 14:00 • 22:00) for automatic refresh.

CONFIG keys (edit inside the Sheet)

KeyDescription
CLICK_THRESHOLDClicks before a keyword or term can be excluded
CTR_HIGH / CTR_LOWBounds that trigger High / Low CTR tips
CPC_MULTIPLIERHigh-CPC flag if Avg CPC > multiplier × account average
CPA_MULTIPLIERHigh-CPA flag if CPA > multiplier × account average

Customise Brand Detection

Edit BRAND_PATTERNS near the top of the script:

const BRAND_PATTERNS = [
  /mybrand/i,      // ASCII
  /ماي براند/i     // Arabic
];
