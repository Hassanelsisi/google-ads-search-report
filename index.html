Google Ads Search-Campaign Reporting Script

This repository provides:

  • search-report.gs – a read-only Google Ads Script that creates an 8-tab Google Sheet.
  • index.html – these docs (renders automatically on GitHub Pages).

Quick Start

  1. Open Google Ads → Tools → Bulk actions → Scripts and paste search-report.gs.
  2. Replace SPREADSHEET_ID with an empty Google Sheet’s ID.
  3. Preview → Authorise → Run once. Schedule to refresh daily.

Customise Brand Detection

Edit BRAND_PATTERNS near the top of the script:

const BRAND_PATTERNS = [
  /mybrand/i,    // ASCII
  /ماي براند/i   // Arabic
];

CONFIG Keys

KeyDescription

CPC_MULTIPLIERHigh-CPC flag if Avg CPC > multiplier × average.
CPA_MULTIPLIERHigh-CPA flag if CPA > multiplier × average.
